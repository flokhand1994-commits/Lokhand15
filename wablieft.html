📥 Artikelen ophalen

Voer het fetch-script eenmalig uit om artikelen te laden.
Je hebt een betaald abonnement op wablieft.be nodig.

Stap 1 — Maak een bestand .env in de projectmap:
WABLIEFT_EMAIL=jouw@email.com
WABLIEFT_PASSWORD=jouwwachtwoord
Stap 2 — Installeer Python-afhankelijkheden:
pip install requests beautifulsoup4 python-dotenv
Stap 3 — Haal de artikelen op:
python fetch_wablieft.py

Herlaad daarna deze pagina. Herhaal stap 3 om de nieuwste artikelen te krijgen.

Tik op een woord voor de vertaling · Tap any word to translate
